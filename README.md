# ZhiPu Translate Chrome Extension

A Chrome extension for web page translation using ZhiPu GLM API.

## Development

1. Open Chrome and navigate to `chrome://extensions/`
2. Enable "Developer mode"
3. Click "Load unpacked" and select this directory

## Features

- Right-click to translate entire page
- Right-click to translate selected text
- Bilingual display (original + translation)
- Multiple target languages support
- Multiple GLM models support (GLM-4, GLM-4-AIR, GLM-4-FLASH)
- Page-level caching

## Configuration

1. Get your API Key from https://open.bigmodel.cn/
2. Click the extension icon or right-click to translate
3. Enter your API Key when prompted
4. Configure language and model in settings
